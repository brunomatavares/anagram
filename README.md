# anagram
Check if two strings are Anagram of each other

Tested with jdk1.8.0_11

## Documentation
* mvn javadoc:javadoc
* anagram/target/site/apidocs/index.html

## Tests
* mvn test

## Install
* mvn install

## Run
* cd target
* java -jar anagram-0.0.1-SNAPSHOT-jar-with-dependencies.jar -s1 "Punishment" -s2 "Nine Thumps"
